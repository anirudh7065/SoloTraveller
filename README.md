# SoloTraveller.in
>This is my personal project inspired by my love for solo travelling and the interest to give back all the knowledge earned through my experience to the solo travellor's community.

### I am building an web application to help people with :-
    - Finding people with same interest.
    - Finding budget friendly and good          accommodation. 
    - Creating a platform to interact with people for advice,discussions and planning about trips.
    - It will also serve as a travel blog for the travellor's .

There are many more ideas coming in the future updates.

This website is gonna be fully open-sourced so that other travel enthusiasts can also contribute to this website after the intial build.

### Website contents :-

    • It will contain login page for which I am currently considering clerk auth.(It will be applied on later updates).
    • About section to learns about the basics of solo travelling, things needed, how to plan things etc.
    • It will have a comment section which will in later updates be converted into blog page.
    •  Place specific planning and everything needed to know that particular place.
